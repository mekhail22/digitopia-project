# Digitopia — Educational App for Kids

An interactive educational web application designed for 5th-grade students, covering **Science** and **Mathematics** topics.

## 🎯 Features
- Interactive science lessons
- Math exercises and quizzes
- Kid-friendly user interface
- Engaging and easy to navigate

## 🛠️ Tech Stack
- Python
- Streamlit

## 🚀 Run Locally
```bash
git clone https://github.com/mekhail22/digitopia-project.git
cd digitopia-project
pip install -r requirements.txt
streamlit run app.py
